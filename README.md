</p>

This project is a quadruped robot named QuadroNav. It is designed to achieve stable walking and turning using a gait-based control system and inverse kinematics (IK). The robot uses servo motors to actuate the leg joints and can be extended with sensors for basic navigation and obstacle avoidance.


<ul>
  <li><strong>Four-legged locomotion:</strong> Stable walking and smooth turning using coordinated leg motion.</li>
  <li><strong>Gait generator:</strong> Produces synchronized step patterns (stance + swing) for all four legs.</li>
  <li><strong>Inverse kinematics (IK):</strong> Converts target foot positions into accurate joint angles.</li>
  <li><strong>Adjustable postures:</strong> Supports different body heights and stances (stand, crouch).</li>
</ul>
